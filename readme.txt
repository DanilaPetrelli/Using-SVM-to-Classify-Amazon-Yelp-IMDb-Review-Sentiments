Description of the four Python scripts:

- char_based.py : Character-based feature selection on the whole corpus. Change n in line 49 to get different ngrams. Comment TfIdf in line 79 to "switch" it on/off.
- wordbased.py : Word-by-word feature selection on the whole corpus.
- separate_sets_word.py : Word-by-word feature selection on the three separate sets of reviews.
- separate_sets_char.py : Character-based feature selection on the three separate sets of reviews.

Uncomment last lines in each script to show predicted sentences.

The folder contains a dataset created for the Paper 'From Group to Individual Labels using Deep Features', Kotzias et. al,. KDD 2015
